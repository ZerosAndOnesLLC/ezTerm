What's inside

Everything you need to manage a fleet of remote boxes.

One window. One vault. Every machine, local or remote, at your fingertips.

⌬

Pure-Rust SSH

Built on russh — tokio-native, no OpenSSL FFI, identical on every OS.

🔒

Encrypted vault

Argon2id KDF and ChaCha20-Poly1305 AEAD on every stored secret. Zeroized in memory on drop.

⇅

SFTP side-pane

Drag-drop files from Explorer. 32 KiB streaming chunks with live byte-accurate progress.

⚡

WSL & local shells

Any WSL distro under ConPTY. cmd, pwsh, or any absolute path with a starting directory.

🖥

X11 forwarding

Tick a box. Remote GUI apps render as native windows. VcXsrv bundled on Windows.

↻

MobaXterm import

Point at your .mxtsessions. Sessions, folders, and private keys imported cleanly.

SFTP

A file browser that lives on your terminal.

Connect once. The SFTP pane opens a second channel on the same session — no second auth, no second password prompt.

  • Breadcrumb navigation, right-click context menu (Download, Rename, Delete)
  • Drag from Windows Explorer straight into the remote directory
  • 32 KiB streaming chunks with live progress bars
  • Drag out to Explorer to download files locally (Windows)
Sessions

Every knob, exactly where you'd expect it.

SSH options live behind a tabbed dialog — General, Terminal, Advanced, Forwards. Sane defaults; deep overrides when you need them.

  • Per-session keepalive, connect timeout, compression
  • Custom TERM, env vars, initial command
  • Local, remote, and dynamic SOCKS5 port forwards
  • X11 forwarding toggle (works out of the box on Windows)
