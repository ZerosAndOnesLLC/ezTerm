ezTerm is a free, open-source SSH client with MobaXterm-style sessions, a built-in encrypted vault, SFTP, WSL tabs, and X11 forwarding. Rust + Tauri. Windows · Linux · macOS. GPLv3.

features

$ what you get out of the box

SSH

russh-powered SSH

Pure-Rust SSH stack. Password, private key, or SSH-agent auth. Host-key TOFU with hard-fail on mismatch.

VAULT

Encrypted credentials

Argon2id KDF → ChaCha20-Poly1305 AEAD. Master password unlocks the app. Secrets zeroized on drop.

SFTP

SFTP side-pane

Docked file browser per tab. Drag-drop upload from Explorer with 32 KiB streaming chunks and live progress.

WSL

WSL + local shells

Any WSL distro under ConPTY. cmd / pwsh / custom paths with optional starting directory.

X11

X11 forwarding

Tick "Forward X11" and remote GUI apps render as native windows. VcXsrv bundled on the Windows release.

IMPORT

MobaXterm import

Point at .mxtsessions or MobaXterm.ini. Sessions + private keys imported into the vault.

FWD

Port forwarding

Local (-L), Remote (-R), or Dynamic SOCKS5 (-D) — saved per session, auto-started on connect.

UX

Familiar workspace

Sessions sidebar with folders + drag-drop. Tabbed terminals with status dots. Split panes, find, scrollback.
